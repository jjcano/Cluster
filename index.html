Scala

Apache Spark

combina un sistema de computación distribuida a través de clusters de ordenadores con una manera sencilla y elegante de escribir programas. ... Spark mantiene la escalabilidad lineal y la tolerancia a fallos de MapReduce, mejora sus bondades gracias a DAG y RDD.

Docker

Permite crear contenedores ligeros y portables para las aplicaciones software que puedan ejecutarse en cualquier máquina con Docker instalado, independientemente del sistema operativo que la máquina tenga por debajo, facilitando así también los despliegues.

Hadoop

Apache Hadoop es un framework de software que soporta aplicaciones distribuidas bajo una licencia libre. Permite a las aplicaciones trabajar con miles de nodos y petabytes de datos.Es un proyecto de alto nivel Apache que está construido en Java.
